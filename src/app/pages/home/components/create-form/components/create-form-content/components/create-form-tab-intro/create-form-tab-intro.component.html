Desde aquí podrás crear contenido para Antic's Code.
¡Es muy sencillo, introduce los datos, escribe el artículo y listo!

Antes de empezar, vamos a darte algunos consejos a seguir para crear Artículos de calidad.
  • Es inprescindible que sepas Markdown ya que los Artículos están escritos en este formato.
  • Si vas a escribir un Artículo damos por hecho que estás familiarizado con lo vas a decir y por lo tanto el Artículo debería ser ameno, no muy largo y fácil de entender para llegar al máximo público posible.
  • No vamos a corregir la gramática del Artículo, por eso intenta escribir el Artículo lo mejor que puedas. El equipo de moderación si puede mirar que no haya contenido inadecuado.
  • Si quieres escribir buenos Artículos, recuerda utilizar todo lo que Markdown ofrece. Listas, citas, bloques de código. Además es importante saber en que orden usar estos elementos.
  • Recuerda que puedes obtener inspiración de Artículos creados por moderadores.
