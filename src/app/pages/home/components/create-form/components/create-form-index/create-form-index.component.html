¿Qué es Markdown?

Markdown nació como herramienta de conversión de texto plano a HTML. Esta herramienta fue creada en 2004, y se distribuye de manera gratuita bajo una licencia BSD.


Aunque en realidad Markdown también se considera un lenguaje que tiene la finalidad de permitir crear contenido de una manera sencilla, y que en todo momento mantenga un diseño legible, para simplificar puedes considerar Markdown como un método de escritura.
